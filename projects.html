PickOne

My first Android App was a fun little game that allowed you to choose between two options (like apples vs bananas). The app would update in real time with other users responses to create a graph with the statistics of every other users votes. I worked on this project in a group with two other UIUC students. Here is the link to the github repository: PickOne App

RPZ Testing Application

I developed this app in order to be used by a client who tests Reduced Pressure Zone (RPZ) devices for local businesses and homes in Naperville. It works as a mobile database for test records that can be easily accessed and updated in the field. You can find the written code on my github linked above.
